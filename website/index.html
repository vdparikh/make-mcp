Get Started Why Make MCP?

Powered by the visual builder

Drag-and-drop interface, multiple execution types, and one-click export. Everything you need to ship MCP servers.

Visual Server Builder

Create MCP servers through a drag-and-drop interface. Define tools, resources, and prompts without YAML or code.

Multiple Execution Types

REST API, GraphQL, Webhooks, CLI (kubectl, docker, terraform), Database, JavaScript, Python, and Visual Flow (pipeline → tool).

Built-in Authentication

API Key, Bearer Token, Basic Auth, OAuth 2.0 with visual configuration on each tool.

Live Testing Playground

Test tools with mock input before deployment. See results in tables or cards.

One-Click Export

Download as Node.js project, Docker image, or push to GitHub. Ready to run in Cursor or any MCP client.

Context-Aware Execution

Auto-inject user identity, permissions, and org data into tool calls.

AI Governance Layer

Policy engine to control tool access: rate limits, allowed roles, time windows, and approvals.

Self-Healing Tools

Auto-detect failures and suggest fixes for your tool configurations.

Server Composition

Combine multiple MCP servers into one and export as a single config.

Security Score

In-app score (0–100%, grade A–F) based on the SlowMist MCP Security Checklist; shown while building and in the marketplace.

Passkey-Only Auth

Sign in with a passkey. No passwords. WebAuthn-based, secure by default.

OpenAPI Import

Turn an OpenAPI spec into an MCP server in one step. Perfect for existing APIs.

QUICK START

Get your MCP server started

Make MCP is a batteries-included toolkit for building and publishing MCP servers. Create tools, resources, and prompts through the UI—then export as Node.js, Docker, or push to GitHub. First-class support for REST, CLI, Flow, and OpenAPI import.

Get started Source code
Terminal
$ git clone https://github.com/vdparikh/make-mcp.git
✔ Cloning repository
$ cd make-mcp && docker-compose up --build
✔ Building backend and frontend
✔ Starting PostgreSQL
✔ Running at http://localhost:3000
$ open http://localhost:3000
✔ Register with email + passkey, create your first server

VISUAL EDITOR

Tool Builder

Visual schema editor with execution types: REST API, GraphQL, Webhooks, CLI (kubectl, docker, terraform), Database, JavaScript, Python, and Visual Flow. Configure auth, input/output schemas, and security hints per tool.

TEST BEFORE DEPLOY

Testing Playground

Test tools with mock input before deployment. Run requests from the UI and see results in raw JSON, or use table and card layouts for a richer preview. CLI tools show a mock indicator so you know what runs in production.

BEST PRACTICES

Security Score

In-app score (0–100%, grade A–F) based on the SlowMist MCP Security Checklist. See which criteria you meet—input validation, tool hints, rate limiting, RBAC—and improve before publishing. The score is shown in the marketplace for published servers.
